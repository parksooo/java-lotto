##로또 게임을 위한 구현 목록

```
유효성 체크 기능
- 엽력받은 money가 1000원으로 나누었을 때 나머지가 있는 경우 v
- 입력받은 money가 1000원미만 일 경우 v
- 입력받은 money가 숫자가 아닐 경우 v
- 당첨 번호가 ,로 이어지지 않았을 경우 
- 당첨 번호가 6자리 초과, 미만일 경우 v
- 입력 받는 모든 번호가 1 ~ 45 의 범위에 들어가지 않았을 경우
- 입력 받는 당첨 번호가 중복일 경우
- 숫자 이외의 문자가 입력 되었을 경우
- 모든 예외 처리는 [ERROR] 메세지로 시작 하면서 예외 처리 
```
```
출력부 기능 V
- "구입금액을 입력해주세요" 출력 함수 v
- 랜덤 로또 번호 출력 함수 v
- "n개를 구매했습니다." 출력 함수 v
- "당첨 번호를 입력해 주세요." 출력 함수 v
- "보너스 번호를 입력해 주세요." 출력 함수 v
- 당첨 통계 출력 함수 v
-  "3개 일치 (5,000원) - n개 v
    4개 일치 (50,000원) - n개 v
    5개 일치 (1,500,000원) - n개 v
    5개 일치, 보너스 볼 일치 (30,000,000원) - n개 v
    6개 일치 (2,000,000,000원) - n개" 출력 함수 v
- "총 수익률은 n%입니다." 출력 함수 v

```
```
랜덤의 로또 번호를 생성하는 기능
- 랜덤 로또 번호를 생성하는 camp.nextstep.edu.missionutils.Randoms의 pickUniqueNumbersInRange()를 활용
- 6개의 번호를 중복없이 List에 추가하고, 오름차순으로 정렬하여 List로 내보낸다.
```
```
입력 받은 당첨 번호와 생성한 로또 번호를 비교하며, 일치하는 갯수 판별 기능
- 랜던 생성한 n개의 번호가 담겨있는 map을 받아서 for문을 돌면서 리스트를 비교한다.
- 비교는 retainall을 사용하여, 변경된 랜덤 로또 번호의 list의 size를 구해서 확인한다.
- size가 3, 4, 5, 6을 판별한다. 
- 보너스 번호가 포함되어있는 list에서 다시 한번 retainall을 사용하여 예외 경우를 찾아낸다
```
```
중복 번호가 3개 이상 있는 리스트 받아와서 금액을 산정하는 기능
- 중복 번호가 3, 5, 6에 대하여 출력 함수과 연동
- 보너스 일 경우에도 출력 함수와 연동
```
```
수익률 정산 해주는 기능
-중복 번호로 얻은 금액을 받아서 쓴 금액과 이익이 난 금액을 산정하여, 소수점 둘째 자리에서 반올림후 출력 함수와 연동
```
